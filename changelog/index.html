Status Tag Input & Appearance Guide

1. Item Change Tags (Input & Result):

Input: [NEW]

✓ NEW Indicates that a new feature, change, or element was added.

Input: [FIXED]

✓ FIXED Indicates that an issue or bug was successfully resolved.

Input: [FIXING]

✖ FIXING Indicates known issue/bug currently being addressed.

Input: [REMOVED]

✖ REMOVED Indicates that a feature, element, or dependency was taken out.

2. Release Status Tags (For New Versions):

These appear next to the version number:

  • ALPHA Early testing, unstable build
  • BETA Feature complete, public testing
  • RELEASE Production ready, stable
